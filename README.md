# Parle Business Performance Analysis Dashboard 2024

## 📊 Project Overview
Interactive Excel dashboard analyzing Parle retail business performance for 2024.  
Built using **Pivot Tables, Pivot Charts, and Slicers** for dynamic filtering by Month, City, and Category.

### Key Insights
- Total Sales: ₹15,62,379
- Total Units Sold: 11,936
- Total Profit: ₹3,12,897
- Average Profit Margin: 19.35%

- Peak sales month and top-performing city/category (add 1-2 quick highlights here)
- Online vs Offline channel comparison
- Payment mode preferences (UPI, Card, Cash, Wallet)

## 🚀 How to Use
1. Download the `.xlsx` file.
2. Open it in Microsoft Excel.
3. Use the **slicers** on the left (Month, City, Category) to filter data interactively.
4. All charts and KPI cards update automatically.

## 📈 Visualizations Included
- Monthly Sales Trendline
- Top Sales by City (Horizontal Bar)
- Top FMCG Categories by Profit Margin
- Payment Mode Preference (Donut Chart)
- Channel Performance Analysis (Revenue vs Units - Offline vs Online)
